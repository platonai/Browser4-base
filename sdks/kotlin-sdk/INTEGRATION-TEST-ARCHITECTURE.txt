# Kotlin SDK Integration Test Architecture

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        Kotlin SDK Integration Tests                      │
└─────────────────────────────────────────────────────────────────────────┘

                              Test Execution Flow
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                           │
│  1. Maven/JUnit starts test                                             │
│                    ↓                                                     │
│  2. @SpringBootTest starts PulsarRestServerApplication                  │
│                    ↓                                                     │
│  3. Server starts on random port (e.g., 52341)                          │
│                    ↓                                                     │
│  4. TestServerConfiguration starts Mock EC Server (port 18080)          │
│                    ↓                                                     │
│  5. Test Base class creates PulsarClient(baseUrl = "http://localhost:52341") │
│                    ↓                                                     │
│  6. Test methods execute SDK operations                                 │
│                    ↓                                                     │
│  7. After test: cleanup sessions and resources                          │
│                    ↓                                                     │
│  8. After all tests: servers shut down                                  │
│                                                                           │
└─────────────────────────────────────────────────────────────────────────┘


                           Component Architecture
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                           │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                     Test Layer                                   │   │
│  │  ┌──────────────────┐  ┌──────────────────┐  ┌──────────────┐ │   │
│  │  │ PulsarClient     │  │ WebDriver        │  │ PulsarSession│ │   │
│  │  │ IntegrationTest  │  │ IntegrationTest  │  │ Integration  │ │   │
│  │  └──────────────────┘  └──────────────────┘  └──────────────┘ │   │
│  │           ↓                      ↓                    ↓         │   │
│  │  ┌────────────────────────────────────────────────────────────┐│   │
│  │  │      KotlinSdkIntegrationTestBase (@BeforeEach/@AfterEach) ││   │
│  │  └────────────────────────────────────────────────────────────┘│   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                 ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                     SDK Layer                                    │   │
│  │  ┌──────────────┐  ┌──────────────┐  ┌───────────────────────┐ │   │
│  │  │ PulsarClient │  │  WebDriver   │  │  PulsarSession        │ │   │
│  │  │              │  │              │  │  AgenticSession       │ │   │
│  │  └──────────────┘  └──────────────┘  └───────────────────────┘ │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                 ↓                                        │
│                          HTTP Requests                                   │
│                                 ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                 REST Server Layer                                │   │
│  │  ┌─────────────────────────────────────────────────────────────┐│   │
│  │  │ PulsarRestServerApplication (Spring Boot, Random Port)      ││   │
│  │  │  - Session Management                                        ││   │
│  │  │  - WebDriver API Controllers                                 ││   │
│  │  │  - AgenticSession API Controllers                            ││   │
│  │  │  - Page Loading & Extraction                                 ││   │
│  │  └─────────────────────────────────────────────────────────────┘│   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                 ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │              Browser Layer (PulsarContext)                       │   │
│  │  - Chrome/Chromium instances                                     │   │
│  │  - Browser automation                                            │   │
│  │  - Page rendering                                                │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                 ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │          Mock EC Server (port 18080)                             │   │
│  │  - Product list pages                                            │   │
│  │  - Product detail pages                                          │   │
│  │  - Test HTML pages                                               │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                           │
└─────────────────────────────────────────────────────────────────────────┘


                            Test Suite Structure
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                           │
│  sdks/                                                                    │
│  │                                                                        │
│  ├── kotlin-sdk/                        # SDK 模块（保持干净）           │
│  │   ├── pom.xml                        # 最小依赖                       │
│  │   └── src/                                                             │
│  │       ├── main/kotlin/               # SDK 源码                       │
│  │       │   ├── PulsarClient.kt                                         │
│  │       │   ├── WebDriver.kt                                            │
│  │       │   ├── AgenticSession.kt                                       │
│  │       │   └── Models.kt                                               │
│  │       └── test/kotlin/               # 仅单元测试                     │
│  │           ├── PulsarClientTest.kt    # No server required            │
│  │           ├── WebDriverTest.kt       # State management only         │
│  │           └── SessionTest.kt         # Model validation              │
│  │                                                                        │
│  └── kotlin-sdk-tests/                  # 独立测试模块                  │
│      ├── pom.xml                        # 包含所有测试依赖               │
│      └── src/test/                                                       │
│          ├── kotlin/ai/platon/pulsar/sdk/integration/                   │
│          │   │                                                            │
│          │   ├── KotlinSdkIntegrationTestBase.kt   # Base class         │
│          │   │   - @SpringBootTest                                       │
│          │   │   - Server lifecycle                                      │
│          │   │   - Client setup/cleanup                                  │
│          │   │                                                            │
│          │   ├── PulsarClientIntegrationTest.kt    # HTTP client tests  │
│          │   │   - Session creation/deletion                             │
│          │   │   - GET/POST/DELETE operations                            │
│          │   │   - Error handling                                        │
│          │   │                                                            │
│          │   ├── WebDriverIntegrationTest.kt       # Browser automation │
│          │   │   - Navigation (navigateTo, currentUrl, reload)          │
│          │   │   - Element interaction (click, fill, type)              │
│          │   │   - Waiting (waitForSelector, exists)                    │
│          │   │   - Scrolling (scrollDown, scrollToBottom)               │
│          │   │   - Content extraction (selectTextAll, extract)          │
│          │   │   - Screenshots (captureScreenshot)                      │
│          │   │   - Script execution (executeScript)                     │
│          │   │                                                            │
│          │   ├── PulsarSessionIntegrationTest.kt   # Page loading       │
│          │   │   - load(), open(), submit()                             │
│          │   │   - normalize() URL handling                             │
│          │   │   - extract() field extraction                           │
│          │   │   - scrape() end-to-end                                  │
│          │   │   - loadAll() batch operations                           │
│          │   │                                                            │
│          │   ├── AgenticSessionIntegrationTest.kt  # AI automation      │
│          │   │   - act() single AI action                               │
│          │   │   - run() autonomous tasks                               │
│          │   │   - observe() page observation                           │
│          │   │   - agentExtract() AI extraction                         │
│          │   │   - summarize() content summary                          │
│          │   │   (Optional, requires AI configuration)                  │
│          │   │                                                            │
│          │   ├── server/                           # Test server config  │
│          │   │   ├── PulsarRestServerApplication.kt                     │
│          │   │   └── TestServerConfiguration.kt                         │
│          │   │                                                            │
│          │   └── util/                             # Test utilities      │
│          │       ├── TestUrls.kt                   # Test URL constants │
│          │       └── TestHelpers.kt                # Helper methods     │
│          │                                                                │
│          └── resources/                                                  │
│              └── application-sdk-integration-test.properties            │
│                                                                           │
└─────────────────────────────────────────────────────────────────────────┘


                          Test Tag Strategy
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                           │
│  @Tag("IntegrationTest")     ← All integration tests                    │
│      │                                                                    │
│      ├─ @Tag("RequiresServer")   ← Needs REST server                    │
│      │                                                                    │
│      ├─ @Tag("RequiresBrowser")  ← Needs Chrome/Chromium                │
│      │                                                                    │
│      ├─ @Tag("RequiresAI")       ← Needs AI/LLM config (optional)       │
│      │                                                                    │
│      ├─ @Tag("Fast")             ← < 5 seconds                           │
│      │                                                                    │
│      └─ @Tag("Slow")             ← > 10 seconds                          │
│                                                                           │
│  Test Execution:                                                         │
│  - SDK unit tests:   cd sdks/kotlin-sdk && mvn test                     │
│  - Integration tests: cd sdks/kotlin-sdk-tests && mvn test -DrunIntegrationTests=true │
│  - Full tests:       cd sdks/kotlin-sdk-tests && mvn test -DrunFullTests=true │
│  - Fast tests only:  mvn test -Dgroups="Fast"                           │
│                                                                           │
└─────────────────────────────────────────────────────────────────────────┘


                        Performance Targets
┌──────────────────────────┬──────────────┬──────────────┬───────────────┐
│ Test Type                │ Target Time  │ Max Time     │ Notes         │
├──────────────────────────┼──────────────┼──────────────┼───────────────┤
│ Single unit test         │ < 1 second   │ 5 seconds    │ No I/O        │
├──────────────────────────┼──────────────┼──────────────┼───────────────┤
│ Single integration test  │ < 10 seconds │ 30 seconds   │ With server   │
├──────────────────────────┼──────────────┼──────────────┼───────────────┤
│ Unit test suite          │ < 10 seconds │ 30 seconds   │ Fast feedback │
├──────────────────────────┼──────────────┼──────────────┼───────────────┤
│ Integration test suite   │ < 5 minutes  │ 10 minutes   │ Full coverage │
├──────────────────────────┼──────────────┼──────────────┼───────────────┤
│ Mock server startup      │ < 5 seconds  │ 10 seconds   │ Background    │
├──────────────────────────┼──────────────┼──────────────┼───────────────┤
│ REST server startup      │ < 10 seconds │ 30 seconds   │ Spring Boot   │
└──────────────────────────┴──────────────┴──────────────┴───────────────┘


                      CI/CD Pipeline
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                           │
│  ┌────────────┐                                                          │
│  │ Git Push / │                                                          │
│  │ Pull Request│                                                         │
│  └──────┬─────┘                                                          │
│         │                                                                 │
│         ├─────────► Unit Tests (Always)                                  │
│         │              - Fast (< 30s)                                     │
│         │              - No server required                               │
│         │              - Build validation                                 │
│         │                                                                 │
│         └─────────► Integration Tests (Conditional)                      │
│                        - On PR to main branch                             │
│                        - On schedule (nightly)                            │
│                        - Manual trigger                                   │
│                        - Before release                                   │
│                        │                                                  │
│                        ├─ Setup JDK 17                                    │
│                        ├─ Install Chrome                                  │
│                        ├─ Build project                                   │
│                        ├─ Run integration tests                           │
│                        ├─ Generate reports                                │
│                        └─ Upload artifacts                                │
│                                                                           │
│  Release Pipeline:                                                       │
│  ┌────────────┐                                                          │
│  │  Release   │                                                          │
│  │  Trigger   │                                                          │
│  └──────┬─────┘                                                          │
│         │                                                                 │
│         ├─────────► Full Test Suite (Required)                           │
│         │              - Unit tests                                       │
│         │              - Integration tests                                │
│         │              - Performance tests                                │
│         │              - Must pass to proceed                             │
│         │                                                                 │
│         └─────────► Deploy to Maven Central                              │
│                                                                           │
└─────────────────────────────────────────────────────────────────────────┘


                    Implementation Phases
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                           │
│  Phase 1: Infrastructure (High Priority) ✓                              │
│  ├─ Create kotlin-sdk-tests module                                       │

│  ├─ Set up test directory structure                                      │
│  ├─ Implement KotlinSdkIntegrationTestBase                              │
│  ├─ Configure PulsarRestServerApplication                               │
│  └─ Configure Maven profiles for test execution                         │
│                                                                           │
│  Phase 2: Basic Tests (High Priority)                                   │
│  ├─ PulsarClient integration tests                                       │
│  │  └─ Session, HTTP methods, error handling                             │
│  └─ Basic WebDriver tests                                                │
│     └─ Navigation and element finding                                    │
│                                                                           │
│  Phase 3: Full Feature Tests (Medium Priority)                          │
│  ├─ Complete WebDriver test suite                                        │

│  ├─ Create utility classes                                               │

│  └─ Configure CI/CD workflows                                            │
```
