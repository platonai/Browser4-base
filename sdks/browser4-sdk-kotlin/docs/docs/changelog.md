# Changelog

All notable changes to Browser4 Kotlin SDK.

## [4.5.0-SNAPSHOT] - 2024-01-30

### Added
- Local driver mode with automatic Browser4.jar download
- `AgenticSession.getOrCreate()` convenience method
- AI-powered browser automation (`act`, `run`, `observe`)
- Intelligent data extraction with schemas
- Page summarization capabilities
- Event streaming for agent operations
- Agent history tracking
- Comprehensive WebDriver API
- Screenshot capture (viewport, full-page, element)
- JavaScript execution support
- Scroll control methods
- CSS selector-based extraction
- One-step `scrape()` method
- Session and driver management
- Page event handlers
- Custom HTTP headers support

### Features

#### Core
- PulsarClient for HTTP communication
- PulsarSession for page loading and extraction
- AgenticSession for AI automation
- WebDriver for browser control
- Automatic session cleanup

#### AI Capabilities
- Natural language actions

#### Data Extraction

#### Browser Control
