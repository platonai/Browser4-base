# Memory For Coworker

Memory 存放在 300logs 下。

四类记忆：

1. 每日记忆，总结当天的任务执行情况，如 MEMORY.20260628.md
2. 月度记忆，总结本月的所有任务执行情况，如 MEMORY.202606.md
3. 年度记忆，总结本年度的所有任务执行情况，如 MEMORY.2026.md
4. 全部记忆，总结 coworker 自开始以来的所有任务执行情况，MEMORY.md

Coworker 需要导入所有记忆，以便在执行任务时参考过去的经验教训，避免重复犯错，并持续改进工作效率和质量。
Coworker 每执行一个任务后，都需要在 MEMORY.xx.md 中总结当天的记忆内容。
月度、年度、全部记忆每日更新一次，Coworker 每次执行都需要检查月度、年度、全部记忆，看看当日是否已经更新，如果没有更新，则立即更新。
