# 实现 browser4-cli 方法

实现 `sdks/skill/SKILL.md` 中的所有方法，提供详细测试。

最终在 `browser4-cli` 项目中实现。

- 如有必要，修改 OpenAPI 定义和后端实现
- 如有必要，修改后端 `WebDriver` 实现（e.g., `PulsarWebDriver`）

提示：

1. After each command, browser4-cli provides a snapshot of the current browser state.
2. The snapshot is YAML representation of the accessibility tree of a page.
3. The YAML file contains a backend id for each accessible element, which can be used for subsequent commands to interact with the element.
4. The cli to interact with an element: `browser4-cli hover e4`, where `e4` is the backend id of the element.
