# Create coworker memory generator script

Currently, we have coworker-daily-memory-generator, which is a script that generates daily memory summaries based on the logs in the 300logs directory. This script will analyze the logs for the current day, extract key information such as tasks performed, issues encountered, solutions implemented, and any important insights gained. The generated memory summary will be saved in the `coworker/tasks/300logs/YYYY/MM/DD/MEMORY.YYYYMMDD.md` file.

We need coworker-memory-generator to generate all the memory summaries.

Memory 存放在 300logs 下。

四类记忆：

1. 每日记忆，总结当天的任务执行情况，如 MEMORY.20260628.md
2. 月度记忆，总结本月的所有任务执行情况，如 MEMORY.202606.md
3. 年度记忆，总结本年度的所有任务执行情况，如 MEMORY.2026.md
4. 全部记忆，总结历史所有任务执行情况，MEMORY.md

注意：

- 全部记忆和年度记忆可以根据月度记忆内容总结，也可以根据每日记忆内容总结。默认是月度记忆内容总结。
